Gettobyte     : Reinventing Tech with Bytes

A Platform To learn firmware development, Device Driver for MCU peripherals, Embedded C and to Write application Codes for Microcontrollers(both 8 bit and 32 bit) by interfacing different Sensors(Arduino Compatible and Industrial Compatible)and Modules( IoT Modules: GPS, ESP, GSM, RF 433, etc) using those Device Driver Libraries. In Short A Complete Gateway from Firmware point of view for building things in IoT, Embedded Systems, Automation,Automotive
